This directory contains CmdBox component for Lazarus IDE, which provides a command line component.

The component that is situated here is not original; it was modified by Kernozhistky Alexander. The changes are:

1. Added a GetText method and Text property to get the text from the command line.
2. Package distribution was re-structured (example directory for usage example, package directory for the package).
3. Added a package version, authors and license to the package config file.
